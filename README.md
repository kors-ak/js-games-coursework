# Мини‑игры
*Учебный проект по frontend-разработке*

### Описание проекта: 
Сборник из 6 простых браузерных игр с интуитивно понятным интерфейсом.  
**Основная задача сайта** — предоставление пользователям увлекательного способа провести время в браузере: развлечься, потренировать внимание и логику, снять стресс. Элементы сайта могут быть полезны как развлекательный раздел на корпоративном портале или личном блоге (повышает вовлечённость посетителей).

### Особенности проекта:

**Адаптивность.** Сайт корректно отображается на мобильных устройствах, планшетах и десктопах.  
**Интерактивные элементы.** Реализованы эффекты наведения и плавные переходы. Каждая игра реагирует на действия пользователя: клики, ввод текста, выбор вариантов.  
**Простота и доступность.** Игры не требуют регистрации, дополнительной загрузки или сложных правил.  
**SEO-оптимизация.** Проект оптимизирован для поисковых систем (семантическая вёрстка).  
**Доступность (Accessibility).** Сайт адаптирован для скринридеров (альтернативный текст для изображений и логическая структура заголовков).

### Разнообразие игровых механик:

**Угадай число** (случайные числа от 1 до 100, подсказки);  
**Простая арифметика** (генерация примеров, проверка ответа);  
**Переверни текст** (обработка ввода, реверсирование строки);  
**Камень, ножницы, бумага** (выбор игрока, логика победы/поражения);  
**Викторина** (вопросы с вариантами ответов, подсчёт баллов);  
**Генератор случайных цветов** (изменение фона по клику).


### Использованные технологии:

HTML5;  
CSS3 (Flexbox, Grid, анимации, адаптивные медиазапросы);  
JavaScript (логика игр, обработка событий, обновление интерфейса);  
SVG (векторные изображения);   
Google Fonts (подключение веб‑шрифтов);  
Git и GitHub для контроля версий и деплоя.
