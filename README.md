# Мини-игры

Учебный frontend-проект, представляющий собой коллекцию из шести браузерных мини-игр, реализованных на чистом JavaScript без использования сторонних библиотек и фреймворков.

## Демо

https://kors-ak.github.io/js-games-coursework/

## О проекте

Проект был создан для практики навыков адаптивной вёрстки и разработки интерактивных пользовательских интерфейсов на Vanilla JavaScript.

В рамках проекта реализованы различные игровые механики, работа со случайными данными, обработка пользовательского ввода и динамическое обновление интерфейса без перезагрузки страницы.

## Функциональность

### Угадай число

Игрок пытается угадать случайное число от 1 до 100, получая подсказки после каждой попытки.

### Простая арифметика

Генерация случайных математических примеров с проверкой правильности ответа.

### Переверни текст

Обработка пользовательского ввода и реверсирование строки.

### Викторина

Серия вопросов с вариантами ответов и подсчётом итогового результата.

### Камень, ножницы, бумага

Классическая игра против компьютера со случайным выбором соперника.

### Генератор случайных цветов

Случайное изменение цвета фона страницы с автоматической настройкой контрастности текста.

## Реализованные возможности

- адаптивная вёрстка для мобильных устройств, планшетов и десктопов;
- семантическая HTML-разметка;
- интерактивные элементы интерфейса;
- обработка пользовательских событий;
- работа со строками, массивами и объектами;
- генерация случайных чисел и значений;
- динамическое изменение содержимого страницы через JavaScript;
- использование SVG-графики;
- базовые принципы доступности (alt-атрибуты, логическая структура заголовков).
